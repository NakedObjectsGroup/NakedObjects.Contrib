NakedObjects.Contrib is a placeholder for projects and code snippets that relate to
the Naked Objects Framework, but do not form part of the framework itself.

Naked Objects Group Ltd has set up this repository as a service to the Naked Objects
user community, and will grant read/write access to anyone who has something to 
contribute. We will simply ask you to confirm that all code you commit is made available
to others under the Apache license.

Please note that Naked Objects Group does not actively vet or manage any code in this 
repository.
